# github 처음 시작할 때
## -1 프로젝트 처음 로드하기
1. 파일탐색기(window키 + R) 누르고 cmd 입력
2. 프로젝트 실행할 파일 생성하기
   - 터미널에서 cd 명령어로 폴더 이동
   - 만약 어떤 폴더가 있는지 모르겠다면 dir 명령어 사용
   - mkdir "파일 이름"으로 프로젝트 담을 파일 생성
   - 아래 git clone 명령어 복사 후 터미널에 붙여넣고 실행
```Bash
git clone https://github.com/seongchan-bae/Gyeongbuk-Tycoon.git
```

3. Unity hub 실행
   - 유니티 버전은 파일과 호환성을 위해 Unity 6 3.18버전으로 실행시켜주세요
4. Project -> add -> add project from disk 실행
5. 프로젝트 정상 실행 확인
